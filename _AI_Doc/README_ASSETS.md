2. **로컬 이미지 추가**
   - 부동산 사진 3장을 다운로드
   - 이 폴더에 `sample_house.jpg`, `sample_house2.jpg`, `sample_house3.jpg`로 저장

3. **Placeholder 이미지 생성**
