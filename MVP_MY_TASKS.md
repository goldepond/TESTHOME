## 📋 일상 운영 작업 (매일/매주)

### 1️⃣ 관리자 대시보드 모니터링

**접속:** `https://배포도메인/#/admin-panel-myhome-2024`

**해야 할 일:**
- [ ] **매일 1회 확인 (오전/오후)**
  - 새로운 견적문의 확인
  - 중개사 응답 여부 확인
  - 통계 대시보드 확인 (총 문의, 대기중, 완료 수)

**예상 소요 시간:** 10-15분/일

---

### 2️⃣ 미가입 중개사 이메일 전송 (수동)

**시나리오:** 판매자가 견적을 요청했는데, 해당 중개사가 플랫폼에 가입되어 있지 않음

**해야 할 일:**
- [ ] **관리자 대시보드 → 견적문의 관리 탭 접속**
- [ ] **견적문의 카드에서 확인:**
  - 중개사명 확인
  - 이메일 주소가 있는지 확인
    - 있음: 다음 단계로
    - 없음: 중개사 정보 조회하여 이메일 주소 입력 필요

- [ ] **"이메일 보내기" 버튼 클릭**
  - 이메일 클라이언트가 자동으로 열림
  - 이메일 내용이 자동으로 생성됨
  - 확인 후 전송

- [ ] **또는 "링크 복사" 버튼 클릭**
  - 링크를 복사하여 직접 전달 (카카오톡, 문자 등)

**예상 소요 시간:** 견적 1건당 2-3분

**참고:** 
- 이메일 내용은 자동 생성됨 (매물 정보, 문의 내용 포함)
- 링크도 자동 생성됨 (중개사가 클릭하면 답변 페이지 접속)

---

### 3️⃣ 중개사 응답 모니터링 (수동 확인)

**해야 할 일:**
- [ ] **매일 1회 확인**
  - 중개사가 답변을 제출했는지 확인
  - 답변 내용 확인 (부적절한 내용 여부 체크)

- [ ] **응답이 없는 경우 (선택적)**
  - 3-5일 후 재연락 고려
  - 또는 다른 중개사 추천

**예상 소요 시간:** 5-10분/일

---

### 4️⃣ 비정상 루트 처리 (수동 확인)

**해야 할 일:**
- [ ] **동일 매물 중복 요청 확인**
  - 같은 주소로 여러 중개사에게 요청한 경우 확인
  - 최대 5곳까지만 허용 (수동으로 판단)

- [ ] **응답 없는 견적 처리**
  - 7일 이상 응답 없는 견적 확인
  - 필요 시 상태를 "만료"로 변경 (수동)

**예상 소요 시간:** 주 1회, 30분
